Racing Track Visualizer

Watch cars navigate famous racing circuits from F1, MotoGP & WEC

General Informations

What is this ?

This is a vite web app which use anime.js to animate car movements on your favorite race tracks all arround the world.

How to use it ?

On the home page select a racing category (Like F1 or WEC), select a track on the card view then enjoy ! (You can pause the or reset car position)

Why this ?

After watching LeMans I remembered the Anime.js site, on the home page there was a very cool effect (with the track in 3d) and I thought “why not reproduce it”? Then finally I found that the RacingCircuit.info site was a bit boring.... If you're going to make a 3D circuit, why not make them all?

QA / FAQ

1 - It will be updated with more tracks and features ?

All my plans are on the Project Backlog on Github, you can see what I plan to do, what I do now and what is done.

2 - How you do the 3D ?

You fool ! It's not 3D, it's just a 2D SVG path with a perspective effect applied to it. You can see the code for it on the "tracks" named files and the RacingTrackVisualizer.js file.

3 - Can I contribute ?

Of course ! You can contribute by adding new tracks and categories (please it could help me so much), fixing bugs (or only tell me your issues if you're not a dev) and even improving the design if you want. Simply go to the issues page on Github and create a new issue or a pull request.
